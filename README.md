# Snake Game

A desktop implementation of the classic Snake Game developed using Python and Tkinter.

## Features

* Real-time keyboard controls
* Dynamic food generation
* Score tracking
* Collision detection
* Persistent leaderboard system
* Replay functionality
* Player name registration

## Technologies Used

* Python
* Tkinter
* File Handling
* Object-Oriented Programming

## Project Structure

```text
Snake-Game/
│
├── snake.py
└── Snake_Leaderboard.txt
```

## Overview

This project recreates the classic Snake Game experience using a graphical user interface. Players control the snake using keyboard inputs, collect food to increase their score, and compete for high scores recorded in a persistent leaderboard.

The application demonstrates game loop implementation, event handling, collision detection, and score management using Python.

## Getting Started

Run the application:

```bash
python snake.py
```
